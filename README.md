## Final Project

**Brain Tumor Detection using YOLOv8**

The final project focuses on detecting and localizing brain tumors from MRI images using the YOLOv8 object detection model. The model was trained on a publicly available MRI dataset and evaluated using standard object detection metrics.

### Performance

| Metric | Score |
|---------|------:|
| Precision | 0.91 |
| Recall | 0.90 |
| mAP@0.5 | 0.95 |
| mAP@0.5:0.95 | 0.79 |
